Projenin amacı:
  • Ev sahiplerine sesli komutlarla evlerini kontrol etme imkanı sağlamak.
  • Akıllı güvenlik sistemleri ile evlerin güvenliğini artırmak.
  • Kapı ve LED kontrolü gibi temel ev işlevlerini uzaktan yönetebilme yeteneği kazandırmak.
  • Isı bilgisi gibi çevresel verileri kullanarak ev sahiplerine konforlu bir yaşam sunmak.

Biz Kimiz? Yalova Üniversitesi Elektrik ve Elektronik Mühendisliği son sınıf öğrencileri olarak, MİO Güvenlik Sistemleri projesini geliştirdik. Misyonumuz, ev güvenliği ve konforunu artırmak için yenilikçi çözümler sunmaktır

Kullanılan teknolojiler: Projemizi oluştururken, Web Programlama alanında bir dizi teknolojiyi kullanarak geliştirdik. HTML ve CSS ile web sayfamızın temel yapısını oluşturduk ve görsel tasarımını şekillendirdik. Ardından, Javascript'i kullanarak elektronik parçalar arasındaki iletişimi sağladık, özellikle ESP32 gibi mikrodenetleyicilerle entegrasyonu gerçekleştirdik. API yöntemlerini kullanarak veri alışverişi sağladık ve gerekli komutları programladık. Ayrıca, Web Speech API aracılığıyla sesli komut sistemi oluşturarak kullanıcıların web sitemizle sesli olarak etkileşime geçmelerini sağladık.
